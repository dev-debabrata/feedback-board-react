# Feedback Board

A simple Feedback Board web app built with **React** and **Tailwind CSS**. The app lets users share short opinions/feedback, view others' feedback, and upvote/downvote entries with optimistic UI updates.

---

## Features

- Create, read, and vote on feedback/opinions.
- Optimistic UI updates for vote actions.
- Server-driven form handling using React 19 hooks (or local action-state fallbacks).
- Accessible form validation and error display.
- Tailwind CSS for utility-first styling (colors match original design using arbitrary hex where appropriate).

---

## Tech stack

- React
- Vite (recommended dev server / build)
- Tailwind CSS (utility-first styling)

---

## Quick start (development)

1. Clone the repo:

```bash
git clone <repo-url>
cd feedback-board
```

2. Install dependencies:

```bash
npm install
# or
# yarn
```

3. Run the dev server:

```bash
npm run dev
```

Open `http://localhost:5173` (or the URL printed by Vite) and you should see the app.
